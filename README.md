# Implementação de Serviços AWS - Desafio DIO

## 📋 Sobre o Desafio

Este projeto faz parte do desafio proposto pela [Digital Innovation One (DIO)](https://www.dio.me/) com o objetivo de demonstrar conhecimento prático em serviços da AWS aplicados à redução de custos operacionais. O desafio consiste em:

- Criar um repositório próprio no GitHub para documentar o projeto
- Selecionar 3 serviços AWS com foco em redução de custos
- Elaborar um relatório técnico detalhando a implementação de cada serviço
- Demonstrar casos de uso práticos aplicados a uma empresa fictícia

## 🎯 Objetivo do Projeto

Demonstrar conhecimento em serviços AWS e sua aplicação prática para otimização de custos empresariais através da escolha estratégica de ferramentas cloud computing na empresa fictícia **Abstergo Industries**.

## 🚀 Serviços AWS Implementados

Este projeto documenta a implementação de 3 serviços AWS estratégicos para redução de custos:

1. **Amazon EC2 Auto Scaling** - Otimização automática de recursos computacionais conforme demanda
2. **Amazon S3 Intelligent-Tiering** - Armazenamento inteligente com otimização automática de custos
3. **AWS Lambda** - Computação serverless eliminando desperdícios com servidores ociosos

## 💡 Resultados Esperados

A implementação dos serviços AWS na Abstergo Industries tem como meta:

- **Redução de custos:** Aproximadamente 55% de economia nos custos de infraestrutura
- **Economia mensal:** Estimada em R$ 85.000,00
- **Escalabilidade:** Ajuste automático de recursos conforme demanda
- **Disponibilidade:** Melhoria na disponibilidade e performance dos serviços
- **Automação:** Otimização automática sem intervenção manual

## 🛠️ Tecnologias e Serviços AWS

- **AWS EC2 Auto Scaling** - Escalabilidade automática de instâncias
- **Amazon S3 com Intelligent-Tiering** - Armazenamento inteligente em nuvem
- **AWS Lambda** - Computação serverless orientada a eventos
- **AWS CloudWatch** - Monitoramento e observabilidade
